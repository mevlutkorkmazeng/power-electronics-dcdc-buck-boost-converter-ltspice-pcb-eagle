# Buck Converter Simulation Plan

In this stage of the project, the first goal is to simulate a basic Buck converter in LTspice.

Target example:

Input Voltage: 12V  
Output Voltage: about 5V  

Main components to use:

- MOSFET
- diode
- inductor
- output capacitor

The simulation will focus on:

- output voltage waveform
- inductor current waveform
- switching behavior
- output voltage ripple

After building the first model, the design will be improved step by step.
