# Project Scope

This project focuses on studying DC-DC power converters, especially **Buck and Boost converters**.

The main idea of the project is to understand how switching converters work and how their behavior can be analyzed using circuit simulation.

In this project, the converter circuits are first designed and simulated using **LTspice**.
After observing the simulation results, the goal is to prepare the project for a simple **PCB layout design**.

## Main Objectives

The main objectives of this project are:

* to understand the working principle of Buck and Boost converters
* to design basic converter circuits
* to simulate the circuits using LTspice
* to observe voltage and current waveforms
* to analyze output voltage ripple
* to prepare the design for a simple PCB layout

## Project Approach

The project is developed step by step.

First, the basic theory of switching converters is reviewed.
Then a Buck converter circuit is designed and simulated in LTspice.

After verifying the simulation results, the project continues with documentation and preparation for PCB layout design.

## Expected Outcome

By completing this project, the behavior of DC-DC converters can be better understood.

The final result of the project will include:

* LTspice simulation circuits
* waveform analysis
* documentation of converter behavior
* preparation for PCB layout implementation
